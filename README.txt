САЙТ-ВИЗИТКА «ТИША» — инструкция для технического специалиста

Содержимое:
- index.html — полностью самодостаточный файл (все стили, скрипты, шрифты и картинки вшиты внутрь). Никаких других файлов и сборки не нужно.

Развёртывание:
1. Положить index.html в корень хостинга (Netlify, Vercel, любой статический хостинг или обычный сервер).
2. Обязательно раздавать по HTTPS.
3. Сервер НЕ должен отправлять заголовок Referrer-Policy: no-referrer или same-origin —
   иначе фоновое YouTube-видео упадёт с «Ошибка 153». Правильно: strict-origin-when-cross-origin
   (или вообще не задавать заголовок — в самом файле уже стоит нужный meta-тег и referrerpolicy на iframe).

Про фоновое видео:
- Фоном играет YouTube-ролик u8WaKpRnYa8 (тизер), без звука, с 4-й секунды, по кругу.
- У ролика в YouTube Studio должно быть включено «Разрешить встраивание» (Allow embedding).
- Звук включается кнопкой внизу слева (автозапуск со звуком браузеры запрещают).
